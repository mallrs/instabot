# instabot
Instgram Bot Coded In python3-instagrapi
___
# How To Use
1. Clone The Reppo
```
git clone https://github.com/mallrs/instabot
```
2. Install instagrapi & colorama
```
pip install instagrapi colorama
```
4. Setup session.json Path In main.py Line 10
```
SESSION_FILE_PATH = os.path.expanduser("YOURPATH/session.json")
```
5. Setup Your Account & Passwd In main.py Line 6, 7
```
USERNAME = "YOUR_USERNAME"
PASSWORD = "YOUR_PASSWD"
```
6. Run main.py
```
python3 main.py
```
___

1. Supports Session Login
